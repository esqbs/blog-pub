线性代数

# 向量

空间中有大小有方向的量

# 张成的空间

一组向量通过线性组合生成的所有向量形成的空间

二维空间中，两个不共线的向量的线性组合生成的所有向量张成的空间是整个二维空间；两个共线的向量的线性组合生成的所有向量张成的空间是一条直线

三维空间中，两个不共线的向量的线性组合生成的所有向量张成的空间是二维平面；三个不共线的向量的线性组合生成的所有向量张成的空间是整个三维平面；

# 线性相关

对于一组向量，当向其中加入一个向量时，它们张成的空间维度增加，这个向量和这一组向量线性无关；否则线性相关

# 秩

一组向量中线性无关的向量的个数

# 基

向量空间中的一组基是张成该空间的一个线性无关向量集

# 线性变换

线性变换保持网格线平行且等距分布，并且保持原点不动

# 矩阵

线性变换后基向量的坐标组成的矩阵可以描述这个线性变换

例：

逆时针旋转90度

剪切

复合变换，逆时针旋转90度，后剪切

# 行列式

行列式代表的是线性变换后单位面积或体积的缩放倍数

单位面积缩放到原来的6倍

单位面积压缩为0，代表空间被压缩到了更小的维度上

单位面积缩放到原来的-3倍，i和j发生了翻转

三维空间中表示单位体积缩放倍数

# 点积

两个向量的点积可以看成是一个向量的模与另一个向量模在当前向量上投影长度的乘积；

还可以看成是将一个向量做线性变换到一维

# 叉积

两个向量的叉积大小为平行四边形的面积，方向垂直，且满足右手定则

# 基变换

将一个向量转换到另一组基下，做线性变换，最后转回原来的那组基下

# 特征值和特征向量

旋转没有特征值

剪切有特征值
