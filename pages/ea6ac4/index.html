css值类型和语法

CSS 属性值定义语法 - CSS（层叠样式表） | MDN (mozilla.org) (opens new window)

# css值类型

CSS值类型文档大全 (opens new window)

shape-outside = none | [ <basic-shape> || <shape-box> ] | <image>                           
<shape-box> = <box> | margin-box  
<image> = <url> | <gradient>  
<box> = border-box | padding-box | content-box

以shape-outside为例，它支持三种不同的数据类型：

  1. <shape-box>
    • <box> ---- background-origin和background-clip等css属性的类型
      • border-box
      • padding-box
      • content-box
    • margin-box
  2. <basic-shape> ---- clip-path和offset-path等css属性的类型
    • inset()
    • circle()
    • ellipse()
    • polygon()
    • path()
  3. <image> ---- background-image和mask-image等css属性的类型
    • <url>
    • <gradient>
    • element()
    • image()
    • image-set()
    • cross-fade()
    • paint()

# css属性值定义语法

css属性值定义语法包含以下3种基本组成元素：

  • 关键字
  • 数据类型
  • 符号

例如：

// to 是关键字
// <angle> <side-or-corner> <color-stop-list> 是数据类型
// [] ? , 是符号
linear-gtadient([<angle> | to <side-or-corner>,]? <color-stop-list>)

  1. 关键字

    • 通用关键字 auto，none等只被部分css属性支持
    • 全局关键字 inherit, initial等被所有css属性支持

  2. 数据类型

  3. 符号

    • 字面符号

      符号 名称 描述
      , 并列分隔符 用来分隔多个并列值
      / 缩写分隔符 在css缩写属性中用于分离类型形同但属于不同css属性的值

    • 组合符号

      符号 名称 描述
      并列 符号为普通的空格字符，表示各部分必须按顺序出现
      && 与组合符 各部分必须出现，但可以不按顺序
      || 或组合符 各部分至少出现一个，但可以不按顺序
      | 互斥组合符 各部分恰好出现其中一个
      [] 方括号 将各部分分组，优先级最高

    • 数量符号

      符号 名称 描述
      无数量符号 恰好出现一次
      * 星号 可以出现任意次
      + 加号 可以出现一次或多次
      ? 问号 可以出现零次或一次
      {A,B} 花括号 出现最少A次，最多B次
      # 井号 可以出现一次或多次，多次出现时必须以逗号分隔
      ! 叹号 表示当前分组必须产生一个值

注意：

  1. 斜杆前后的数据类型一定相同或部分相同

    // 斜杆前是background-position,后是background-size
background: 0 / 0;

  2. 井号后可指定数量范围

    // 指定范围
rgba(<number>#{3}, <alpha-value>?)
// 不指定范围
box-shadow: none | <shadow>#

  3. 叹号表示当前分组必须产生一个值

    // 表示<image-src>和<color>至少出现一个
<image()> = image(<image-tags>? [<image-src>? , <color>?]!)

# css全局关键字

关键字 描述
inherit (opens new window) 继承，可以应用于任何 CSS 属性，包括 CSS 简写 all (opens new window)。IE8开始支持。
initial (opens new window) 把css属性的计算值还原成css语法中规定的初始值。
unset (opens new window) 如果这个属性本来有从父级继承的值（这个属性默认可以继承，且父级有定义），则将该属性重新设置为继承的值，如果没有继承父级样式，则将该属性重新设置为初始值。
revert (opens new window) 将样式还原成浏览器内置样式

应用：

  1. inherit重置输入框的内置字体，子元素实现高度继承，背景图像继承；display:initial是inline。

    input,textarea {
    font-family: inherit;
}
p {
    height: inherit;
	background-image: inherit;
}
p {
    display: inntial;
}

# 指代所有属性的all属性

CSS (opens new window) all 简写属性 (opens new window) 将除了 unicode-bidi (opens new window) 与 direction (opens new window) 之外的所有属性重设至其初始值，或继承值。

应用：

  1. all:unset 可以让任意一个元素样式表现和<span>一样

  2. all:revert 可以让元素恢复成浏览器默认的样式

    /* 仅IOS Safari有效*/
@supports (-webkit-overflow-scrolling: touch) {
    progress {
        all: revert;
    }
}
