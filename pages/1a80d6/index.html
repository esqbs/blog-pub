是否支持flash

传统浏览器可以使用window.ActiveXObject检查浏览器是否启用相关的控件。

谷歌、火狐、微软Edge、Safari等现代浏览器不支持ActiveXObject，它们支持navigator.plugins检查浏览器插件。

function hasUsableSWF(){
    var swf;     
    if(typeof window.ActiveXObject != "undefined"){
        swf = new  ActiveXObject("ShockwaveFlash.ShockwaveFlash");
    }else{
         swf = navigator.plugins['Shockwave Flash'];
    }
    return swf ? true : false;
};

上面的hasUsableSWF检查浏览器是否支持flash，如果支持返回true， 否则返回false。
