css渐进增强处理技巧

Can I use (opens new window)

# 直接使用css新特性

  • 视觉表现相关css属性：border-radius、box-shadow、text-shadow、filter等
  • 交互体验增强css属性：scroll-behavior、overscroll-behaivor等
  • 性能增强css属性：will-change等

# 利用属性值的语法差异实现兼容

加载效果，IE10+使用png背景图外加animation属性，linear-gradient IE10+支持；IE9及以下使用gif背景。

.icon-loading {
    display: inline-block;
    width: 30px;
    height: 30px;
    /* 所有浏览器识别 */
    background: url('./loading.gif');
    /* IE10+浏览器识别,覆盖上一行 */
    background: url('./loading.png'), linear-gradient(transparent, transparent);
    animation: spin 1s linear infinite;
}

@keyframes spin {
    0% {
        transform: rotate(0);
    }
    100% {
        transform: rotate(360deg);
    }
}

下拉浮层，IE9+使用阴影，IE8使用border边框实现。

.panel-x {
    /* 所有浏览器识别 */
    border: 2px solid #ddd;
    /* IE9+浏览器识别,覆盖上一行 */
    border: 1px solid rgba(0, 0, 0, 0);
    box-shadow: 2px 2px;
}

去除inline-block元素间的空白间隙，又能保持空格特性。

.space-size-zero {
    font-size: 0.1px;
    font-size: -webkit-calc(1px - 1px);
}

背景纹理。"#rrggbbaa" (opens new window)，"background-blend-mode" (opens new window)， IE/Edge18及以下不支持#rrggbbaa。

.background-pattern {
    background: floralwhite url(../images/pattern.png) 10px 10px;
    background: repeating-linear-gradient(floralwhite,floralwhite 10px,lightcoral 10px,lightcoral 15px,floralwhite 15px,floralwhite 20px,lightcoral 15px,lightcoral 40px,floralwhite 40px,floralwhite 45px,lightcoral 45px,lightcoral 50px,floralwhite 50px,floralwhite 80px),repeating-linear-gradient(90deg,floralwhite,floralwhite 10px,lightcoral 10px,lightcoral 15px,floralwhite 15px,floralwhite 20px,lightcoral 15px,lightcoral 40px,floralwhite 40px,floralwhite 45px,lightcoral 45px,lightcoral 50px,floralwhite 50px,floralwhite 80px),#00000000;
    background-blend-mode: multiply;
}

# 借助伪类或伪元素区分浏览器

IE9+可以使用伪元素before、after、selection，可以使用伪类checked、disabled 可用

_::before,
.red {
    color: red;
}

IE10+可以使用伪类required、optional、valid、invalid 可用

_:required,
.red {
    color: red;
}

IE11+可以使用伪元素backdrop，::backdrop (opens new window)， "::backdrop" (opens new window)

_::backdrop,
.red {
    color: red;
}
_::-ms-backdrop,
.red {
    color: red;
}

Edge12+可以使用@supports 可用

Edge13+可以使用伪类in-range、out-of-range

区分浏览器类型 慎用

仅火狐，::-moz-progress-bar (opens new window)

/*仅火狐*/
_::-moz-progress-bar,
.red {
    color: red;
}

现代浏览器，:default (opens new window)

_:default,
.red {
    color: red;
}

webkit浏览器，:is() (:matches(), :any()) (opens new window)

:-webkit-any(_),
.red {
    color: red;
}
