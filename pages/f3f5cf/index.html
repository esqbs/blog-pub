# 性能指标

# 核心网页指标

核心网页指标主要衡量网页的加载、互动和视觉稳定性

  • Largest Contentful Paint (LCP) (opens new window) ：最大内容绘制，测量加载性能。为了提供良好的用户体验，LCP 应在页面首次开始加载后的2.5 秒内发生。
  • Interaction to Next Paint (INP) (opens new window)：衡量与网页进行的每个点按、点击或键盘互动的延迟时间，并根据互动次数选择网页最差（或接近最长的互动延迟时间）作为单个代表性值，以描述网页的整体响应能力。
  • Cumulative Layout Shift (CLS) (opens new window) ：累积布局偏移，测量视觉稳定性。为了提供良好的用户体验，页面的 CLS 应保持在 0.1. 或更少。

# 其他指标

  • First Contentful Paint (FCP) (opens new window)：首次内容绘制，衡量从网页开始加载到网页内容的任何部分在屏幕上呈现的时间。
  • Time to Interactive (TTI) (opens new window)：可交互时间，衡量的是从网页开始加载到视觉呈现、其初始脚本（若有）已加载且能够快速可靠地响应用户输入的时间。
  • First Input Delay (FID) (opens new window) ：首次输入延迟，测量交互性。为了提供良好的用户体验，页面的 FID 应为100 毫秒或更短。
  • Total bloking Time (TBT) (opens new window)：总阻塞时间，衡量 FCP 和 TTI 之间的总时长，在该时间段内，主线程处于阻塞状态的时长足以阻止输入响应。
  • Time To First Byte (TTFB) (opens new window)：首字节时间，衡量网络使用资源的第一个字节响应用户请求所需的时间。
